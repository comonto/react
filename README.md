# My React App (MUI + Auth + Protected Routes + Forms)

SPA in React con Material UI, autenticazione base, routing e validazione form.

## Stack
- React + Vite
- Material UI (MUI)
- Zustand (auth state)
- React Router DOM
- React Hook Form + Yup
- TanStack Query (pronto per API)

## Avvio

```bash
npm install
npm run dev
```
